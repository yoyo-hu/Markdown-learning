# github的基本操作

## 常用的英文按键解释

### repository 

<u>仓库</u>：一个项目对应一个仓库

### star

<u>收藏</u>

### fork

<u>复制收藏项目</u>，即

+ 把其他用户的仓库复制一份过来并且带标识**forked from 用户名/仓库名**
+ 仓库是独立存在的（即该仓库修改，原仓库不会被修改）

### pull request

<u>发起请求</u>->即把更新动作发给来源仓库->若来源仓库的用户觉得不错可以合并到来源仓库中

### watch

<u>关注</u>

对象可以是用户也可以是项目->更新时会再首页通知

### issue 

<u>事务卡片</u>

发现项目有bug，与对方建立讨论
